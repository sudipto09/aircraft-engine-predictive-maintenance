# Aircraft Engine Predictive Maintenance

Machine learning project to predict Remaining Useful Life (RUL) of turbofan aircraft engines using NASA CMAPSS dataset.

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## Project Structure
- notebooks/ : analysis and experiments
- src/ : reusable code
- results/ : plots and metrics

## Dataset
NASA CMAPSS turbofan engine dataset
